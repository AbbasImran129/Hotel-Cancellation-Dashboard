# üè® Hotel Booking Data Analysis

This project focuses on analyzing hotel booking data to understand the reasons behind **high cancellation rates**.  
The entire analysis was performed using **Microsoft Excel**, involving data cleaning, pivot table analysis, and dashboard creation to extract actionable business insights.

## üìå Project Objective

The main goal of this project is to identify the **key factors influencing booking cancellations** and to provide insights that can help hotel management reduce the overall cancellation rate.

## üß≠ Process Overview

### **Step 1 ‚Äî Data Cleaning**
- Removed duplicates and handled missing or inconsistent values.  
- Standardized column names and formatted dates for analysis.  
- Filtered out invalid or incomplete booking entries.  
- Ensured numerical and categorical data were properly aligned for pivot analysis.

### **Step 2 ‚Äî Data Analysis using Pivot Tables**
- Created multiple **Pivot Tables** to explore relationships between:
  - Lead time and cancellation rate  
  - Room type and booking status  
  - Deposit type and customer behavior  
  - Guest type (Single, Couple, Family) and cancellation patterns  
- Used filters, calculated fields, and percentages to make comparisons.

### **Step 3 ‚Äî Dashboard Creation**
- Designed an interactive Excel **dashboard** combining visuals such as:
  - Bar charts (for cancellations by lead time and room type)  
  - Pie charts (for cancellation distribution by customer type)  
  - Line graphs (for monthly booking and cancellation trends)  
- Added slicers for dynamic filtering (e.g., by hotel type, month, or customer segment).

### **Step 4 ‚Äî Insight Generation**
- Analyzed the dashboard visuals to identify core issues driving cancellations.  
- Summarized the most influential variables affecting booking outcomes.

---

## üìä Key Insights

- **High lead time** bookings (booked far in advance) had the **highest cancellation rates**.  
- **No-deposit** bookings were much more likely to be canceled than prepaid ones.  
- **Repeat guests** were significantly more reliable, showing lower cancellation rates.  
- **Family and couple** bookings showed different behavioral patterns ‚Äî families tended to book longer stays but canceled more frequently.  
- **Peak seasons** saw higher cancellations, possibly due to overbooking or pricing fluctuations.
